# 🏨 The Golkonda Hotels Clone <span style="float:right;">[![LinkedIn](https://img.shields.io/badge/LinkedIn-0077B5?style=flat&logo=linkedin&logoColor=white)](https://www.linkedin.com/in/syedahmedhasanx3/) [![Portfolio](https://img.shields.io/badge/Portfolio-Live-00df9a)](https://syedahmedx3.github.io/Portfolio/)</span>
A high-fidelity frontend recreation of the official **Golkonda Hotels** landing page. This project focuses on luxury aesthetics, smooth user interactions, and functional booking UI components.

## ✨ Features

* **📱 Fully Responsive:** Seamless experience across mobile, tablet, and desktop devices.
* **🧭 Smooth Navigation:** Interactive menu with refined scroll behavior.
* **🎭 Dynamic UI:** Smooth CSS transitions and keyframe animations for a premium feel.
* **🎠 Image Sliders:** Integrated carousels to showcase hotel suites and amenities.
* **📅 Booking Integration:** Functional **jQuery UI Datepickers** and pop-up messages for a realistic reservation flow.
* **🛠️ Clean Architecture:** Organized file structure following modern web development practices.

## 🚀 Technologies Used

* **HTML5 & CSS3:** For semantic structure and advanced layout styling.
* **JavaScript & jQuery:** Powering the logic for UI interactions and animations.
* **jQuery UI:** Utilized for the date picking functionality and specialized widgets.

## 📂 Project Structure

```text
The-Golkonda-Hotels-Clone               
├─ visuals/             # High-quality images and icons
├─ index.html           # Main entry point
├─ jquery-3.7.1.js      # Core jQuery library
├─ jquery-ui.min.css    # jQuery UI styling
├─ jquery-ui.min.js     # jQuery UI widgets
├─ script.js            # Main interaction logic
├─ style.css            # Custom layout and animations
└─ README.md            # Project documentation
```

## 🛠️ Installation & Setup

1.  **Clone the repository:**  

    ```bash
    git clone https://github.com/syedahmedx3/the-golkonda-hotels-clone.git
    ```
2.  **Navigate to the project directory:**

    ```bash
     cd the-golkonda-hotels-clone
    ```
3.  **Open the project:**
* Simply open index.html in your browser or use the Live Server extension in VS Code.
